# websocket的连接状态

## 状态

| 状态                                   | value                                              |
| -------------------------------------- | ------------------------------------------------- |
| CONNECTING                 | 0 |
| OPEN                               | 1                                          |
| CLOSING  |                                                   2|
| CLOSED               | 3                  |
## 方法
- send
- close WebSocket.close([code[, reason]])
> code 一个数字状态码，它解释了连接关闭的原因。如果没有传这个参数，默认使用1005。

## 事件
- open
- close
- error
- message
