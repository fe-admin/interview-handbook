# 生成器 generator
## Generator
```js
function * gen(){
	yield 1;
	yield 2;
	yield 3;
	return 0;
}
```
## 返回值是Iterator
> 调用 Generator 函数后，该函数并不执行，返回遍历器对象
调用遍历器next方法返回{value, done}对象
执行完再次执行，返回{value: undefined, done: true}

## 与 Iterator 接口的关系
```js
const myIterable = {};
myIterable[Symbol.iterator] = gen;
[...myIterable]// 1,2,3

```
