# promise async await
## Promise
### 特点
- 不受外部影响
> pendding fulfilled rejected
- 状态一变不会在变
### 缺点
- 无法取消Promise，一旦新建它就会立即执行，无法中途取消。
- 如果不设置回调函数，Promise内部抛出的错误，不会反应到外部。
- pending状态时，无法得知目前进展到哪一个阶段

### all race allSettled any
- all 所以成功才成功，一个失败即失败
- race 一个成功就返回
- allSetled 所有成功、所有失败
- any 一个成功就成功，所有失败才失败
