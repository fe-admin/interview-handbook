# 实现下列布局

> 容器 1，2，3，4 宽度相同，容器之间间距为 20px

| 这是一个容器 1 |     | 这是一个容器 2 |     | 这是一个容器 3 |     | 这是一个容器 4 |
| -------------- | --- | -------------- | --- | -------------- | --- | -------------- |


```css
.box {
  display: flex;
  justify-content: space-between;
  flex-wrap: wrap;
}
.item {
  min-width: calc((100% - 60px) / 4);
  max-width: calc((100% -60px) / 4);
}
```
