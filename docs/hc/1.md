# CSS 优先级

| 选择器                         | 描述   |
| ------------------------------ | ------ |
| !important                     | 最高级 |
| 内联                           | 1000   |
| ID                             | 0100   |
| 类选择器/属性选择器/伪类选择器 | 0010   |
| 元素选择器/伪元素选择器        | 0001   |
| 关系选择器/通配符              | 0000   |
