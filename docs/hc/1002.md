# CSS常见的长度单位有哪些？区别是什么？


- px 
> 是一个虚拟长度单位，是计算机系统的数字化图像长度单位，如果px要换算成物理长度，需要指定精度DPI(Dots Per Inch，每英寸像素数)
- em 相对单位，相对于当前元素的字体大小
- %  相对单位，相对于父元素
- rem 相对单位，相对于html中设置的字体大小
- vw/vh
> - 1.vw：1vw等于视口宽度的1%。
> - 2.vh：1vh等于视口高度的1%。
> - 3.vmin：选取vw和vh中最小的那个。
> - 4.vmax：选取vw和vh中最大的那个。
