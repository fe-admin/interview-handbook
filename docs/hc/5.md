# CSS + > ~ 选择器

```html
<div class="box">
    <p>p1</p>
    <p>p2</p>
    <h1>h1</h1>
    <h2>h2</h2>
    <div>
      <h2>h2</h2>
      <p>p3</p>
      <p>p4</p>
    </div>    
</div>
```
- 后代选择器 .box p {}
> 所有后代
- 子选择器 .box > p {}
> 直接后代
- 相邻兄弟选择器 h1 + h2 {}
> 只是选择紧跟着的兄弟元素
- 后续兄弟选择器 p ~ p{}
> 选择所有符合条件的兄弟元素
